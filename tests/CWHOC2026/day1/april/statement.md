## 题目描述

**本题有多组数据。**

下指标求和是一种强大的技术，运用多项式连续点值平移可以做到 $O(\sqrt{n}\log n)$。

但是作为 CWH 的向往者，不能止于这样简单的问题。

你需要解决下指标最大公因数问题。

给定 $n,m$，求出：

$$
\gcd\left(\binom{n}{1},\binom{n}{2},\cdots,\binom{n}{m-1},\binom{n}{m}\right)=\gcd_{i=1}^m\binom{n}{i}
$$

的值。

## 输入格式

从文件 `april.in` 中读入数据。

第一行两个整数 $n,m$。

## 输出格式

输出到文件 `april.out` 中。

输出一行四个两位小数，表示答案。

{{ sample(1) }}

## 数据范围

对于 $100\%$ 的数据：$1 \le n \le 10^6$, $0 \le m \le 10^6$, $1 \le a_i$, $p_i \le n$。

对于 $40\%$ 的数据：$n, m \le 10^3$。

对于 $60\%$ 的数据：$n, m \le 10^5$。

对于另外 $20\%$ 的数据：$m = 0$。
